# Ender Contract 末影契约

末影契约是[明日方舟](https://ak.hypergryph.com/)([Arknights](https://www.arknights.global/))危机合约玩法在MC中的精神移植，灵感来自于<https://www.bilibili.com/video/BV1ST4y1f7KM/>。

原版本（[天灾合约](https://www.bilibili.com/video/BV1ST4y1f7KM/)）已不再使用通关游戏作为目标，末影契约的意义就在于保留该玩法，同时添加一些自己的创意和玩梗。

重点是使用**服务器插件实现**，在拓展更多可能的词条的同时，也使得服务器可以很方便的集成，而不影响原玩法。
